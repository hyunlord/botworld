- Tiles scaled from 16x16 to 32x32 (nearest-neighbor) and assembled into `terrain-sheet.png`

## Building Sprites

**Procedural Top-Down Pixel Art** (generated by `scripts/generate-building-sprites.js`)
- 15 POI building sprites in top-down perspective
- Color palette derived from Kenney Tiny Town aesthetic
- Variable sizes per building type (64x64 to 128x96)
- Buildings: tavern, marketplace, blacksmith, library, temple, farm, mine_entrance,
  inn, watchtower, guild_hall, fountain, ruins, port, fishing_hut, witch_hut

## Procedural Assets

Some tiles in the terrain spritesheet are procedurally generated colored placeholders
created by the `scripts/generate-terrain-sheet.js` build script. These will be replaced
with proper pixel art assets as the project matures.
